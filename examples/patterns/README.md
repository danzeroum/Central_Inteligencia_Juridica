# examples/patterns — demonstrações de padrões de agentes

Exemplos **genéricos** dos padrões de agentes (execução de tarefas, raciocínio,
memória, colaboração, resiliência). São didáticos e não acoplados ao domínio
jurídico.

Para demos **específicas do projeto** (A2A, MCP discovery, treino, multi-agente),
veja os scripts em [`../`](../).

# BuildToFlip v6.1 - Guia Completo de Agentes

## 🎯 Visão Geral
BuildToFlip v6.1 implementa todos os padrões de design agêntico priorizando valor imediato e baixo risco.

### Capacidades Core
1. **Execução Segura**: Sandbox dedicado para todas as ferramentas.
2. **Consenso Inteligente**: Votação ponderada com base em expertise histórica.
3. **Planejamento Adaptativo**: Replanejamento automático após falhas.
4. **Memória Inteligente**: Persistência com esquecimento e compressão.
5. **Autonomia Progressiva**: Agentes ganham confiança com feedback contínuo.
6. **Aprendizado Contínuo**: Experimentação A/B e roteamento inteligente.

## 🔐 Segurança First
- **Nunca** executar ferramentas fora do sandbox.
- **Sempre** validar entradas contra padrões perigosos.
- **Sempre** envolver humanos em ações críticas de alto impacto.

# scripts/dev — utilitários de desenvolvimento

Scripts auxiliares que **não** fazem parte da aplicação em produção.

| Arquivo | Para quê |
|---|---|
| `export_openapi.py` | Gera `docs/API/openapi.json` a partir do app FastAPI (fonte da verdade). |
| `fase2-resolve.py`, `fase2-stubs.py`, `fase2-tests.py` | Artefatos do processo de evolução (FASE 2). Mantidos por histórico; não são executados pela aplicação. |
